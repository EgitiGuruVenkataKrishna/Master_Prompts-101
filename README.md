# Prompt Design Philosophy

All prompts in this repository follow a structured format:

```
Act as a [ROLE]

Objective:
[GOAL]

Context:
[BACKGROUND]

Inputs:
[USER DATA]

Constraints:
[RULES]

Output Format:
[EXPECTED OUTPUT]
```

This structure improves:

* reasoning quality
* response accuracy
* structured outputs
