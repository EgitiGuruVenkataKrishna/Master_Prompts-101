# Common Prompt Mistakes

Avoid these common mistakes.

---

## 1. Being Too Vague

Bad:

Explain AI.

Good:

Explain **machine learning for beginners with examples**.

---

## 2. Missing Context

Always provide context.

Example:

I am a **student preparing for a programming exam**.

---

## 3. No Output Structure

Specify format.

Example:

Provide answer as:

* bullet points
* summary
* table

---

## 4. Too Many Instructions

Keep prompts clear and focused.

---

## 5. No Role Assignment

Example:

Act as a **business strategist**.

Roles improve responses.
