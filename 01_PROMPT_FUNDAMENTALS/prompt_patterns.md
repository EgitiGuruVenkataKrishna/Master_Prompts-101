# Common Prompt Patterns

These are reusable prompt styles.

---

## Role Prompt

Act as a **[ROLE]**

Example:

Act as a **data scientist**.

---

## Instruction Prompt

Tell the AI exactly what to do.

Example:

Generate **10 startup ideas for college students**.

---

## Step-by-Step Prompt

Ask the AI to reason step by step.

Example:

Explain the solution **step by step**.

---

## Output Format Prompt

Specify structure.

Example:

Return the result as a **table**.

---

## Comparison Prompt

Example:

Compare **Python vs Java** for beginners.

---

## Brainstorm Prompt

Example:

Generate **20 creative business ideas**.
