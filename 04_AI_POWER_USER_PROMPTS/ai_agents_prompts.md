# AI Power User Prompts

These prompts use structured reasoning to improve answer quality.

They are useful when solving:

• complex problems
• research questions
• analytical tasks

---

# 1. Step-by-Step Reasoning Prompt

Use this when solving complex problems.

Prompt Template:

Act as an expert problem solver.

Problem:
[PROBLEM]

Instructions:

1. Carefully analyze the problem
2. Break it into smaller parts
3. Solve each part step-by-step
4. Combine results to reach the final answer

Constraints:

* Explain reasoning clearly
* Avoid skipping steps

Output:

Step-by-step reasoning followed by the final answer.

---

# 2. Multi-Perspective Analysis Prompt

Use this to evaluate a topic from different viewpoints.

Prompt Template:

Act as an expert analyst.

Topic:
[TOPIC]

Analyze the topic from three perspectives:

1. Technical perspective
2. Business perspective
3. Societal impact

Output:

Provide structured analysis for each perspective.

---

# 3. Deep Research Prompt

Use this when exploring a new topic deeply.

Prompt Template:

Act as a research analyst.

Research Topic:
[TOPIC]

Goal:
Provide a structured understanding of this topic.

Output structure:

1. Overview
2. Key concepts
3. Important developments
4. Current challenges
5. Future outlook

---

# 4. Critical Thinking Prompt

Use this to evaluate claims or arguments.

Prompt Template:

Act as a critical thinker.

Claim:
[CLAIM]

Tasks:

1. Analyze the claim
2. Identify assumptions
3. Evaluate supporting evidence
4. Highlight potential weaknesses

Output:

Balanced evaluation of the claim.
