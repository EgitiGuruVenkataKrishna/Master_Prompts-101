# Output Format Library

Specifying the output format helps produce **clear and structured responses**.

Below are common formats used in prompts.

---

## Table Format

Use when comparing items or organizing structured data.

Example Instruction:

Return the result in a table.

Example:

| Idea | Problem | Solution |

---

## Bullet Point Format

Useful for summaries and lists.

Example Instruction:

Summarize the key points using bullet points.

---

## Step-by-Step Format

Used for tutorials or problem solving.

Example Instruction:

Explain the solution step by step.

---

## Structured Report

Used for analysis or research tasks.

Example Instruction:

Return the output using this structure:

1. Introduction
2. Key Findings
3. Analysis
4. Conclusion

---

## Checklist Format

Useful for workflows.

Example Instruction:

Provide a checklist for completing this task.
