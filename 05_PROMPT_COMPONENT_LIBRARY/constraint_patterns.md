# Constraint Patterns

Constraints help control how the AI responds.

They improve relevance and reduce unnecessary information.

---

## Length Constraints

Example instructions:

Limit the response to 200 words.

Provide a concise explanation.

---

## Audience Constraints

Example instructions:

Explain the concept for beginners.

Explain this for an advanced technical audience.

---

## Style Constraints

Example instructions:

Use simple language.

Write in a professional tone.

---

## Content Constraints

Example instructions:

Include real-world examples.

Avoid unnecessary technical jargon.

Focus only on the main ideas.

---

## Output Constraints

Example instructions:

Return the result as a table.

Provide numbered steps.

Generate exactly 5 ideas.
