# Prompt Playbook

This playbook shows **practical prompt workflows** that combine multiple prompt techniques.

These examples demonstrate how prompts from this repository can be used in **real scenarios**.

Compatible with:

* ChatGPT
* Claude
* Gemini
* Copilot

---

# 1. Research Workflow

Goal: Understand a new topic deeply.

Step 1 — Topic Exploration

Act as a research assistant.

Topic:
[TOPIC]

Task:
Identify the main areas related to this topic.

Output:
List of key research areas.

---

Step 2 — Deep Explanation

Act as a domain expert.

Research Area:
[AREA]

Task:
Explain this area clearly.

Output:

Overview
Key concepts
Important developments

---

Step 3 — Insight Summary

Act as an analyst.

Task:
Summarize the key insights about this topic.

Output:

* main ideas
* future trends
* important challenges
